Gpt
1.Buatkan kerangka program Java berbasis console dengan nama class PenjumlahanNilaiKelompok dan gunakan Scanner untuk input data dari user.
2.Tambahkan fitur untuk meminta user memasukkan jumlah data terlebih dahulu, lalu simpan dalam variabel integer.
3.Buat dua array berdasarkan jumlah data tersebut:
Array untuk menyimpan nilai
Array untuk menyimpan kode kelompok setiap nilai
4.Gunakan perulangan for untuk meminta user memasukkan:
Nilai ke-i
Kode kelompok dari nilai tersebut
Simpan masing-masing input ke dalam array yang sesuai.
5.Buat perulangan do-while yang memungkinkan user memasukkan kode kelompok yang ingin dihitung total nilainya, dan beri pilihan angka 0 untuk keluar dari program.
6.Gunakan perulangan untuk mengecek setiap data, jika kode kelompok sama dengan yang dipilih user maka jumlahkan nilainya ke dalam variabel total.
7.Jika tidak ada data dengan kode kelompok yang dipilih, tampilkan pesan "Kelompok tidak ada!".
Jika ada, tampilkan total nilai kelompok tersebut.
8.Jika user memasukkan angka 0, hentikan perulangan dan tampilkan pesan "Program selesai".
